# 前端首頁
1. 搜尋功能   -> 搜尋相關文章並顯示在文章展示區
2. 文章展示區 -> 點擊文章跳轉到blog詳情頁面
3. 分類展示區 -> 點擊分類跳轉到分類頁面
4. 標籤展示區 -> 點擊標籤跳轉到分類頁面
